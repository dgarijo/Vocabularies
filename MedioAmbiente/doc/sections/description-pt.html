3. Otalex: Descrição de volta a índice

Extremadura em Espanha e Alentejo e Centro, em Portugal, são regiões de diferentes países, mas com uma ampla gama de interesses comuns. Eles são territórios dentro forntera e continuidade territorial marcado, que partilham características ecológicas, socioeconômicas e ambientais semelhantes.

Iniciado em 2011 como uma continuação para OTALEX II projeto, o projeto OTALEX C "Centro de Monitoramento Territorial e Ambiental Alentejo Extremadura", financiado pelo Programa Operacional de Cooperação Transfronteiriça Espanha Portugal 2007-2013 (POCTEP), visa criar um sistema de gestão e controlo ambiental através da consolidação de OTALEX IDE (Spatial Data Infrastructure of-Extremadura-Alentejo Observatório Territorial Center) como uma plataforma para o intercâmbio de informações ea cooperação institucional entre as administrações no Alentejo-Extremadura -Center [ ATLAS OTALEX ].

A fim de facilitar um vocabulário comum para a troca de informações foi desenvolvido ontologia aqui descrito. Este foi consultado fontes de dados disponíveis eo atlas OTALEX C [ ATLAS OTALEX ]. Os termos são descritos em detalhe na secção seguinte.
