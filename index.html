Vocabularies by dgarijo

Repository for storing all the different vocabularies and ontologies I have created or contributed to.

View project on GitHub

The Open Provenance Model for Workflows (OPMW)

OPMW is an OPMV, PROV and P-plan extension to model the executions and definitions of scientific workflows. The official OPMW URI is http://www.opmw.org/ontology/

P-Plan Vocabulary

P-Plan is a PROV extension for linking Plans and parts of plans to their respective executions. The official P-Plan URI is http://purl.org/net/p-plan

Workflow Fragment Description Ontology

P-Plan extension to represent workflow fragments and their relationships to workflow templates

Workflow Invocation Ontology

Wf-invoc is a simple profile of the P-plan ontology to describe how workflow steps are invoked within a workflow execution.

Workflow Motif Vocabulary

Ontology for describing Workflow Motifs. Workflow Motifs outline the kinds of data-intensive activities that are observed in workflows (data-operation motifs) and the different manners in which activities are implemented within workflows (workflow-oriented motifs).

El Viajero

OPMO profile for a News and Blogs scenario. Extension for modeling travels and routes. The official URI for the Ontology is http://webenemasuno.linkeddata.es/ontology/OPMO

The Research Object Model

Family of ontologies that can be used together to define a Research Object. The family of ontologies is composed of four main ontologies: RO, Wfdesc, Wfprov and Roevo

The Research Object Optimization Extension

RO-Optimization is an extension of the RO model in order to be able to describe ooptimizations of workflows and their results.

The Video Game Ontology

The Video Game Ontology is aimed at modelling video game related information. The main goal is to capture knowledge about events that happen in video games and information about players.

The PROV-O Ontology

W3C standard for interchanging provenance on the Web.

OntoSoft

Vocabulary for people to publish their software in order for it to be accessible to other users, allow others to run it, cite it, understand it, and use it in their experiments.
